# Préambule

Le CSS, acronyme de Cascading Style Sheets, est un langage utilisé pour définir la présentation visuelle des documents HTML. Il permet de séparer le contenu (HTML) de la mise en forme (design), ce qui améliore la lisibilité du code et facilite la maintenance.

Le CSS fonctionne en appliquant des règles de style aux éléments HTML. Une règle CSS est composée d’un sélecteur (qui cible un ou plusieurs éléments) et d’un bloc de déclarations (contenant des propriétés et leurs valeurs).

Les principaux rôles du CSS incluent :

* gérer les couleurs, les polices et les tailles de texte
* contrôler la disposition (layout) des éléments
* ajouter des espacements (marges, padding)
* créer des animations et des transitions
* rendre les pages adaptatives (responsive design)

Le CSS repose sur les notions de cascade et de spécificité. Plusieurs règles peuvent s’appliquer à un même élément, et certaines ont priorité sur d’autres en fonction de leur précision et de leur ordre d’apparition.
